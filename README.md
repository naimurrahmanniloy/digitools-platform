
  # 🚀 Digital Platform

  **Perfect use of AI for Self-Improvement**
  

---

### 📖 About
This digital Platform will help the user to improve their skills. It is the tool which will help on work.

### 🛠 Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Notifications:** React-Toastify
- **Icons:** Lucide React
- **Framework:** Daisy UI

### ✨ Key Features
*   **🎯 Feature One:** Supercharge the Digital Workflow with the help of AI. Added the services with API and called them using Async function, done optional chaining to access the data. Successfully previewed on the UI.
*   **⚡ Feature Two:** Can added the services and it will showed on the cart section. It can be purchased. After clicking a notiffication will popped up.
*   **📱 Feature Three:** After adding the services it will be showed on the cart which on the nav section also the toggling button will preview the purchased courses. After clicking on the checkout button the cart will be empty and it will show a popup again.
